When we first made changes to the repo, the changes haven't yet been committed locally. This will show up when we enter the command "git status".

Any files with changes will show us being not staged for commit. Unless we add them using the git add command

'Git add .' will add all files with changes to be comitted, alternatively 'git add <filename>' will add the specific files.
